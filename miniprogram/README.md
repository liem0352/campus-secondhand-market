## 快速三步

1. 先启动后端：`code/family_finance` → `python manage.py runserver 0.0.0.0:8000`
2. 微信开发者工具 → 导入本目录 `code/miniprogram` → **不使用云服务**
3. **详情 → 本地设置** → 勾选 **不校验合法域名…** → **编译** → 登录 `zhangsan` / `123456`

## 默认接口地址

`app.js` → `apiBase: 'http://127.0.0.1:8000/api'`（仅模拟器；真机改局域网 IP）

## 启用语音插件（可选）

若模拟器报 **「插件未授权使用」** 或需要语音记账，须同时满足：

1. 在 `app.json` 根级增加：

```json
"plugins": {
  "WechatSI": {
    "version": "0.3.5",
    "provider": "wx069ba97219f66d99"
  }
}
```

2. 登录 [微信公众平台](https://mp.weixin.qq.com) → 设置 → 第三方设置 → **插件管理** → 添加「微信同声传译」。
3. 使用**已绑定该小程序**的开发者微信扫码登录开发者工具。

**默认仓库已去掉 `plugins` 声明**，便于测试号/未授权 AppID 直接编译运行；手动记账不受影响。
