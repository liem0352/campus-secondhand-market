## 1. 文档体系说明

### 1.1 项目概况

校园二手交易平台（Campus Market，下称 CM）是面向高校学生的二手物品线上交易系统，**一后端 + 三前端**架构：

- **后端**：Django 4.2 + DRF + SimpleJWT + MySQL 9.4（位于 `d:\文件\工作 作业\微信小程序实训\4次课程内容\综合实训\backend\market\`）
- **买家端**：微信小程序（原生框架 + 自定义 tab-bar）
- **卖家端**：Vue 3.5 + TypeScript + Element Plus + ECharts（端口 3000）
- **平台管理端**：Vue 3.5 + JavaScript + Element Plus（端口 3001）

> 项目业务 Spec 见 [pivot-to-secondhand-market/spec.md](file:///d:/文件/工作 作业/微信小程序实训/4次课程内容/综合实训/.trae/specs/pivot-to-secondhand-market/spec.md)。

### 1.2 业务范围

| 业务域 | 范围 |
|--------|------|
| 用户体系 | 注册、登录、JWT 鉴权、校园身份认证、信用分（0-100）、徽章 |
| 商品 | 发布、审核、上下架、状态机（draft → pending → on_sale → pending_sold → sold）、分类、收藏 |
| 交易 | 订单状态机（requested → confirmed → shipping → completed）、评价、信用分联动 |
| 社交 | 私聊会话、消息、未读数 |
| 平台治理 | 举报、处理、审计日志、信用分调档、封禁 |
| AI 能力 | 一键发布、议价参考价、内容审核、文案润色、议价话术、关键词提取、智能客服 |

### 1.3 对标产品

- **闲鱼 App**：商品状态机、私聊、收藏、举报的 UX 参考
- **得物 / 小红书**：瀑布流 / 双列布局的视觉参考
- **微信小程序原生**：自定义 tab-bar、rpx 适配、wx.login

### 1.4 差异化能力

- **AI 一键发布**：多模态识别图片自动生成标题 / 描述 / 价格建议（`/api/ai/publish-assist/`）
- **议价参考价**：根据类目 + 成色 + 学校 + 季节，AI 输出建议价区间（`/api/ai/price-suggest/`）
- **智能议价话术**：卖家授权后由 AI 协助议价（`/api/ai/negotiate/`）
- **校园身份认证**：以学校 + 学号作为可信基础，结合信用分双向治理
- **三端协同**：单一后端 + 小程序买家 + Web 卖家工作台 + Web 平台管理后台

### 1.5 课程基线

- 与 `pivot-to-secondhand-market/spec.md` v1.0 对齐；
- 课程共 **4 次实训**，每次对应 docs/ 中一组核心文档；
- 实操类文档（`QUICKSTART.md` / `部署说明.md` / `联调检查清单.md` / `实验指导书.md`）已与本索引同步发布，不在重写范围。
