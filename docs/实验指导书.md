### 2.1 数据模型

```python
class Product(models.Model):
    seller = ForeignKey(User, on_delete=CASCADE)
    category = ForeignKey(Category, on_delete=PROTECT)
    title = CharField(max_length=80)
    description = TextField()
    price = DecimalField(max_digits=10, decimal_places=2)
    condition = CharField(choices=CONDITION_CHOICES)   # 全新 / 9 成新 / ...
    status = CharField(choices=STATUS_CHOICES, default='pending_review')
    school = CharField(max_length=64)
    view_count = IntegerField(default=0)
    favorite_count = IntegerField(default=0)
    created_at = DateTimeField(auto_now_add=True)

class ProductImage(models.Model):
    product = ForeignKey(Product, related_name='images', on_delete=CASCADE)
    image = ImageField(upload_to='products/%Y/%m/')
    sort = IntegerField(default=0)
```

### 2.2 商品状态机

```
pending_review  ──审核通过──>  on_sale
                            │
                            ├── 有人下单 ──> pending
                            │
                            └── 卖家下架 ──> offline

pending        ── 订单完成 ──> sold
```

### 2.3 AI 一键发布流程

```
[拍照/选图]
    ↓ wx.uploadFile
[/api/upload/image/]  →  返回 image_url
    ↓
[/api/ai/publish-assist/]  →  调用 LLM 多模态
    ↓
{category, title, description, suggested_price, confidence}
    ↓
[自动填充表单] → 用户调整 → 发布
```

**降级策略**：`LLM_API_KEY` 未配置或 LLM 调用失败时，返回 mock 数据，前端以"AI 推荐"灰色标识区分。

### 二、核心知识点

### 2.1 订单状态机

```
         买家提交"我想要"
                ↓
           [requested]  ← 通知卖家（RabbitMQ）
                ↓ 卖家确认（自取 / 快递）
           [confirmed]  ← 通知买家
                ↓ 双方标记完成
           [completed]  ← 触发评价 + 信用分变更
                ↑
   任意阶段 ← [cancelled]
```

### 2.2 信用分规则

| 事件 | 信用分变动 |
|------|-----------|
| 注册 | 初始 80 |
| 好评（5 星） | +1 |
| 中评（3-4 星） | 0 |
| 差评（1-2 星） | -1 |
| 信用分 < 60 | 发布商品自动进入 `pending_review` |
| 管理员手动调整 | ±N（写入 `AuditLog`） |

### 2.3 私聊与会话

```python
class Conversation(models.Model):
    user_a = ForeignKey(User, related_name='+')
    user_b = ForeignKey(User, related_name='+')
    product = ForeignKey(Product, null=True, blank=True)
    last_message_at = DateTimeField()
    UNIQUE = (user_a, user_b, product)

class Message(models.Model):
    conversation = ForeignKey(Conversation, related_name='messages')
    sender = ForeignKey(User)
    content = TextField()
    image = ImageField(null=True)
    created_at = DateTimeField(auto_now_add=True)
    is_read = BooleanField(default=False)
```

**进入会话默认招呼语**：`你好，我对这件商品感兴趣`

### 2.4 RabbitMQ 异步通知

订单状态变更时，后端发布消息到队列 `market.order.notify`：
- `requested` → 通知卖家
- `confirmed` → 通知买家
- `completed` → 通知双方

消费者（可选 Worker）发送小程序订阅消息 / WebSocket 推送。

- **Web 管理后台 7 页面**（仪表盘 / 用户 / 审核 / 分类 / 举报 / 日志 / AI 配置）

### 2.3 生产部署架构

```
用户 → Nginx (443) → /api/* → Waitress/Gunicorn → Django
                  → /        → Vue 构建产物 (dist)
                  → /media/* → 商品/头像图片
                  → /ws/     → WebSocket（可选）
                                            ↓
                                          MySQL 9.4
                                            ↓
                                       RabbitMQ（订单异步）
```

**组件职责**：

| 组件 | 职责 |
|------|------|
| **Nginx** | 静态文件 / 反向代理 / HTTPS / 负载均衡 |
| **Waitress（Windows）/ Gunicorn（Linux）** | 运行 Django，多进程处理请求 |
| **MySQL** | 持久化存储 |
| **RabbitMQ** | 订单状态变更 / AI 任务 / 推送的异步队列 |
